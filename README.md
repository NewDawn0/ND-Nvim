# nvim-config
My personal neovim configuration

## Installation
```bash
mv ~/.config/nvim ~/.config/nvim.bak # Back up your config
git clone https://github.com/NewDawn0/nvim-config.git ~/.config/nvim # Install config
nvim +PackerSync # Install packages
```
## Updating
```bash
cd ~/.config/nvim
git pull
nvim +PackerSync
```
